# To-Do List Application

A simple command-line to-do list application written in Ruby.

## Features

- Add new tasks
- View all tasks with completion status
- Mark tasks as done
- Delete tasks
- Simple text-based interface

## Requirements

- Ruby (any recent version)

## Usage

Run the application:

```bash
ruby to-do.rb
```

Follow the on-screen menu to manage your tasks:

1. Add a task - Enter a new task description
2. View tasks - Display all tasks with their status
3. Mark task as Done - Select a task number to mark as completed
4. Delete a task - Remove a task by its number
5. Exit - Quit the application

## Example

```
--- To-Do List ---
1. Add a task
2. View tasks
3. Mark task as Done
4. Delete a task
5. Exit
Choose an option: 1
Enter task: Buy groceries
Task added.

--- To-Do List ---
1. Add a task
2. View tasks
3. Mark task as Done
4. Delete a task
5. Exit
Choose an option: 2
1. [ ] Buy groceries
```
